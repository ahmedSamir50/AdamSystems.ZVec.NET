# ZVec.NET.Samples.Console (.NET 10)

```bash
# Interactive menu (recommended)
dotnet run --project samples/ZVec.NET.Samples.Console

# CLI shortcuts still work
dotnet run --project samples/ZVec.NET.Samples.Console -- basics
dotnet run --project samples/ZVec.NET.Samples.Console -- status
dotnet run --project samples/ZVec.NET.Samples.Console -- rag seed-fixtures
```

Commands: `status`, `models`, `basics`, `rag …`, `search …`, `recommend …`, `ask`, `help`, `quit`.

Suggested numbered prompts appear before ask/search/recommend when you omit the query text.
